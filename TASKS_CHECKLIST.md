Tasks Creation

#1 Creating a task with mandatory fields populated only
    - Click "Create Task" button
    - Enter task title (required field)
    - Click "Save" button
    - Check that the task exists on page (enter the title in the Search field)

#2 Creating a task with all fields populated
    - Click "Create Task" button
    - Enter task title, Description, Due on, time, Priority, Client, Assigned to, upload a file
    - Click "Save" button
    - Check that the task exists on page (enter the title in the Search field)
    - Open the Task and verify the input fields

Tasks Editing

#1 Completing a task
    - Checking the checkbox next to task
    - Ensuring that the checkbox turned green
    - Ensuring that by searching by filter "Completed" the task is displayed

#2 Uncompleting a task
    - Checking the checkbox next to task
    - Ensuring that the checkbox turned white
    - Ensuring that by searching by filter "Incomplete" the task is displayed

#3 Updating a task's title
    - Opening a task
    - Updating the title
    - Clicking "Save" button
    - Ensuring the task has the updated name

#4 Updating a task's fields
    - Opening a task
    - Updating the custom field
    - Clicking "Save" button
    - Ensuring the task has the updated field value

#5 Deleting some of a task's fields' values
    - Opening a task with custom fields populated
    - Deleting the value in the custom field
    - Clicking "Save" button
    - Ensuring the field does not have a value anymore

#6 Check that a task can be viewed (opened) after changing its status to Completed/Incompleted

Tasks Search

#1 Filter by status works
    - When selecting "Incomplete", only tasks with status "Incomplete" are displayed
    - When selecting "Completed", only tasks with status "Completed" are displayed
    - When selecting "All", all tasks are displayed

#2 Filter by assignee works
    - When selecting "All", all tasks are displayed
    - When selecting particular assignee, only tasks assigned to this person are displayed
   
#2 Filter by date works
    - When selecting "Custom", custom tasks are displayed
    - When selecting "Due date", tasks that are due soon are displayed
    - When selecting "Date created", tasks are filtered by creation order (ascending)
    - When selecting "Priority", tasks are filtered by priority (descending)

#4 All three Filters work together and do not break each other
    - Combination of several or all filters results in relevant display
